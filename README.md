# 📝 ForIt Challenge - Task Manager

Aplicación web de lista de tareas desarrollada como parte del challenge de ingreso a Academia ForIT 2026.

Permite crear, visualizar, actualizar y eliminar tareas, además de filtrarlas por estado.

---

## 🚀 Tecnologías utilizadas

### Frontend
- React (con Vite)
- Bootstrap
- Fetch API

### Backend
- Node.js
- Express
- SQLite3

---

## ▶️ Ejecutar el backend

- Ir a la carpeta backend:
`cd backend`
- Instalar dependencias:
`npm install`
- Iniciar el servidor:
`npm run dev`
- El backend se ejecuta en:
`http://localhost:3000`

---

## ▶️ Ejecutar el frontend
- Abrir otra terminal y ejecutar:

`cd frontend
npm install
npm run dev`

- El frontend se ejecuta en:
`http://localhost:5173`

---

## 🧩 Funcionalidades

- Crear tareas
- Editar tareas
- Eliminar tareas
- Marcar tareas como completadas o pendientes
- Filtrar tareas:
  - Todas
  - Pendientes
  - Completadas

---

## 💾 Persistencia

Las tareas se almacenan en una base de datos SQLite (`tasks.db`).

---

## 🎨 Estilos

Se utilizó Bootstrap para el diseño de la interfaz.

---

## 🌱 Flujo de trabajo con Git

Se utilizó una estructura basada en Git Flow:

- `main` → versión estable
- `develop` → integración
- `feature/*` → desarrollo de funcionalidades
